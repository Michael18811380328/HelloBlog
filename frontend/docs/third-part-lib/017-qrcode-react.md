# 017 qrcode.react

## 用途

react 的组件，可以生成二维码

## 可靠性

25000 星号，30万下载

## 官网链接

https://github.com/zpao/qrcode.react

https://www.npmjs.com/package/qrcode.react

## 基本使用

```js
var React = require('react');
var QRCode = require('qrcode.react');

React.render(
  <QRCode value="http://facebook.github.io/react/" />,
  mountNode
);
```
